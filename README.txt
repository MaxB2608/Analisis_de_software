##README##

*Por razones que desconozco, "client" se subió vacía. su contenido está en "cliente", donde está el contenido de "client", se recomienda al descargar la carpeta, eliminar client y cambiar el nombre de cliente a client.
* Se pide utilizar en una terminal un "cd client" y posterior a esto utilizar "npm install" y luego para ejecutar la página, insertar "npm start"
* Posterior a lo anterior, se pide utilizar en una terminal un "cd server" (para estar dentro de la carpeta del servidor) y posterior a esto utilizar "npm nodemon" y luego para ejecutar la base de datos, insertar "npm run start". En caso de no funcionar, escriba "npm run dev", a priori el anterior es mejor por temas de velociad.
